# 🏨 Hotel Management Dataset (10K+ Rows)

## 📌 Overview

This dataset simulates a real-world hotel management system, including bookings, customers, payments, rooms, and staff.

It is designed for:

* Data analysis
* SQL practice
* Machine learning experiments
* System design case studies

---

## 📊 Dataset Features

* 5,000+ rows
* Fully normalized schema
* Realistic relationships
* Multiple entities (Hotels, Rooms, Bookings, Payments)

---

## 🧱 Tables

* Hotels
* Rooms
* Customers
* Bookings
* Payments
* Staff

---

## 🔗 Relationships

* One hotel → many rooms
* One customer → many bookings
* One booking → one payment
* One hotel → many staff

---

## 💡 Use Cases

* Booking prediction
* Revenue analysis
* Customer segmentation
* Fraud detection
